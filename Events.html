Lil Texas 日本公演

@ageHa Tokyo

Lil Texas Virtual Set (w/TCPT)

2021年11月13日 16:00~21:00

数々世界のトップアーティストが訪れたageHa TokyoにLil TexasがVirutal Setで出演！また、ageHaは2022の2月には営業終了してしまうということで、とても貴重なイベントとなった。五つのステージでCONNECTアーティストにより音楽が流れていて、いろんな音に出会え、大盛り上がりのイベントだった。

MESTO 日本公演

@WARP SHINJUKU

Mesto Virtual Set (w/TCPT)

2021年11月4日 16:00~22:00

日本で最高峰クラスのクラブ、WARP SHINJUKUで世界的に有名なMESTOがVirutal Setで公演！Cosmo RoomにはCONNECTのアーティストが多く出演し、スペシャルゲストのYuto Hiroseが実力を見せつけた。また、運営メンバーでもあるHarmiはメインステージ出演し、MESTOと共演することができた、CONNECTに取っては大事なイベントだった。

CONNECT FESTIVAL VOL.2

@Roppongi Cube

CONNECT FESTIVAL Vol.2

2021年5月4日 13:00~18:00

ROPPONGI CUBEというクラブで、2回目のCONNECTイベントが開催。今回はDJ MAGとDJANE MAGの二つの部門で日本一位として選ばれたNATSUMIや、Jersey Crew、INDOOR RADIOをスペシャルゲストとして迎えた。

AFTERMOVIEを見よう！

CONNECT FESTIVAL Vol.1

@444QUAD大宮

CONNECT FESTIVAL Vol. 1

2021年3月21日 13:00~20:00

444quad大宮というクラブで、初めてのCONNECTイベントが開催。コロナ禍の中で協力しあい、安全に楽しめるイベントにできたことが一番の誇りである。小さいクラブではあったが、参加者は大盛り上がりで最高のイベントだった。
